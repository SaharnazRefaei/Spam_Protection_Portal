## Quick Start

````Run backend
# open new terminal
# cd project directory
# cd backend
# enter commands below:
$ npm install
$ npm start

```Run frontend
# open new terminal
# cd project directory
# cd frontend
# enter commands below:
$ npm install
$ npm start
# open Google Chrome on http://localhost:3000


````
